配方排列（3×3）：
- 附魔书 如履平地/愈伤/滋养/活力：
	A E A
	B C B
	A D A
	A=主材料×4（兔子脚/恶魂之泪/蒲公英或兰花/糖），B=附魔金苹果×2，C=下界之星，D=书，E=尖塔/肋骨/牧民/涡流盔甲纹饰。

- 附魔书 护佑：
	A E A
	C B C
	A D A
	A=金苹果×4，B=下界之星，C=下界合金锭×2，D=书，E=坚守盔甲纹饰。

- 附魔书 虚无：
	A E A
	C B C
	A D A
	A=末影之眼×4，B=追溯指针，C=紫水晶碎片×2，D=书，E=幽静盔甲纹饰。
