# rks_lite 数据包（v2.3）
## 主要特性
- **附魔扩展**：
- 轻盈（马铠/狼铠/挽具，1级）：↑坐骑移速0.5，跳跃高度0.5，飞行速度0.05

- 恶魔契约（全身护甲，1级，排斥神圣恩典）：↑攻速+0.2、攻击力+1.5、触及范围+0.25、移速+0.01；↓最大生命值-4、幸运-0.5。

- 神圣恩典（全身护甲，1级，排斥恶魔契约）：↑最大生命值+2、护甲韧性+2、幸运+1、移速+0.005、移动效率+0.2、水中移动效率+0.2，摔落伤害倍率-15%，着火时间-0.15；↓攻击力-1。

- 护佑（盾牌或图腾手持，1级）：提供伤害减免和常驻抗性提升1。

- 滋养（头盔或图腾手持，1级）：周期性获得饱和效果。

- 愈伤（胸甲或图腾手持，1级）：常驻生命恢复1。

- 活力（护腿或图腾手持，1级）：移速+0.04、移动效率+0.2，水中移动效率+0.2。

- 如履平地（护腿，1级）：最大行走高度+0.5，跳跃力度+0.5。

- 熔岩舞者（靴子，2级，排斥草径行者）：免疫岩浆块，着火时间-0.2，将脚边的完整熔岩变为岩浆块（不可逆变化）。

- 草径行者（靴子，2级，排斥熔岩舞者）：免疫岩浆块、仙人掌、浆果丛，将脚边的泥土变为草方块（不可逆变化）。

- 太空漫游者（靴子，3级，排斥深海探索者）：重力+X, X=0.01-0.032×等级.(玩家基础重力为0.8)，跳跃力度+0.2。

- 干涸撕咬（剑/斧/矛，2级，排斥火焰附加）：命中施加6秒缓慢X, X=等级×2。

- 无限 Ultra（弓/弩，1级，OP）：所有箭矢与烟花无限。

- 天行者（靴子，1级，OP）：免疫岩浆块、仙人掌、浆果丛，跳跃或空中行走时在脚下生成樱花树叶（按潜行时不生效）。

- 大运（矛类，1级，OP）：移速+2.5、移动效率+1、水中移动效率+1。

### 可合成的附魔书（工作台）
配方排列（3×3）：
- 如履平地/愈伤/滋养/活力：
	A E A
	B C B
	A D A
	A=主材料×4（兔子脚/恶魂之泪/蒲公英或兰花/糖），B=附魔金苹果×2，C=下界之星，D=书，E=尖塔/肋骨/牧民/涡流模板。
- 护佑：
	A E A
	C B C
	A D A
	A=金苹果×4，B=下界之星，C=下界合金锭×2，D=书，E=坚守模板。

### 生存模式暂无法获取的附魔
- 大运、无限 Ultra、天行者：需通过指令/创造模式取得。
